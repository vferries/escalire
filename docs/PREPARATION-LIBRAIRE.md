# À préparer avant notre rendez-vous — administration du site

Ce document est pour toi : il liste ce qu'il faut préparer **avant** notre
rencontre pour que la mise en route de l'espace d'administration du site
(escalire) prenne moins d'une demi-heure. Le mode d'emploi au quotidien est
dans `GUIDE-ADMIN.md` — on le parcourra ensemble.

## 1. Créer ton compte GitHub (~10 min)

Le site est hébergé sur GitHub ; ton compte (gratuit) sert de clé d'accès
sécurisée à l'espace d'administration. Il n'y a rien d'autre à installer.

1. Aller sur https://github.com/signup
2. **Email** : utilise ton adresse professionnelle de préférence.
3. **Nom d'utilisateur** : simple et reconnaissable (ex. `prenom-escalire`) —
   il sera visible dans l'historique des modifications du site.
4. **Mot de passe** : long et unique ; note-le dans ton gestionnaire de
   mots de passe (ou un carnet gardé en lieu sûr, à défaut).
5. Valide le code reçu par email pour activer le compte.

## 2. Activer la validation en deux étapes (2FA) — obligatoire (~5 min)

C'est la protection qui évite qu'un tiers modifie le site avec ton mot de
passe volé.

1. Une fois connectée : photo de profil (en haut à droite) → **Settings** →
   **Password and authentication** → **Two-factor authentication** → Enable.
2. Le plus simple : l'option **Authenticator app** avec une application
   gratuite sur ton téléphone (Google Authenticator, Microsoft
   Authenticator…). L'option SMS marche aussi.
3. **Important** : GitHub affiche des « recovery codes » à la fin —
   imprime-les ou recopie-les, ils permettent de récupérer le compte si
   tu changes de téléphone.

## 3. M'envoyer ton nom d'utilisateur

Un simple message avec ton nom d'utilisateur GitHub suffit — je préparerai
l'invitation avant le rendez-vous.

## 4. À chercher d'ici là

- **Le logo en version vectorielle** (fichier `.svg`, `.ai` ou `.eps`,
  probablement chez le graphiste qui l'a créé) — il rendrait le logo du site
  parfaitement net sur tous les écrans, et plus léger.
- **2 ou 3 coups de cœur** pour étrenner la rubrique : le livre (l'ISBN,
  le numéro à 13 chiffres au dos, nous suffit) + un petit mot de 2-3 phrases.
- **Les prochaines rencontres / dédicaces** déjà programmées (dates, et
  affiches si tu les as).
- **Les horaires exceptionnels** à venir (fermetures d'été, jours fériés).
- Si tu veux : **photos de l'équipe** pour la section
  « Derrière le comptoir ».

## 5. Le jour du rendez-vous

Viens avec **ton téléphone** (nécessaire pour la connexion 2FA). Ensemble,
en moins de 30 minutes :

1. Tu acceptes mon invitation (reçue par email).
2. Première connexion à l'espace d'administration.
3. Tu publies ton premier coup de cœur — chronomètre en main, moins de
   5 minutes entre la saisie et la mise en ligne.
4. Tour rapide du reste : rencontres, horaires, équipe, textes.

En cas de souci avant le rendez-vous (email de validation qui n'arrive pas,
2FA récalcitrante…), appelle-moi, on regarde ça ensemble.
